# blackcardrevokedcli
A command line version of BlackCardRevoked. This version is single threaded.

# Setup
mvn install

# Test
mvn test

## To run the project:
---------
Option 1
---------
java -cp target/BlackCardRevoked-0.0.1-SNAPSHOT.jar main.java.app.App

---------
Option 2
---------
java -jar target/BlackCardRevoked-0.0.1-SNAPSHOT.jar
